# Advanced-Digital-System-Design
Course Projects related to EECE.5500 Advanced Digital System Design

## Project 1: Physical Unclonable Functions
Physical unclonable functions (PUFs) have been proposed as a mechanism to generate CSPRNGs ( Cryptographically-Secure Random Number Generator). In this project we will simulate and implement a type of PUF: A ring oscillator-based physical unclonable function (ROPUF). Using SPICE simulation, we'll simulate the effects of process variations and implement a PUF on an Intel MAX 10 FPGA.
